Cara Mudah Belajar HTML & CSS untuk Pemula

Syvra Tambun

4 September 2025

HTML dan CSS adalah dasar dari pembuatan website. HTML mengatur struktur konten, sedangkan CSS mengatur tampilan dan styling. Untuk pemula, mulailah dengan mempelajari tag-tag dasar HTML seperti heading, paragraph, dan link.

Setelah memahami HTML, lanjutkan dengan belajar CSS untuk membuat website yang menarik. CSS memungkinkan kita mengatur warna, font, layout, dan animasi.

Contoh HTML sederhana:

<h1>Judul Website</h1>
<p>Ini adalah paragraf pertama.</p>
<a href="#">Ini adalah link</a>
Contoh CSS sederhana:

h1 { color: blue; }
p { font-size: 16px; }
a { text-decoration: none; }

Tips: Praktek secara konsisten dan jangan takut bereksperimen. Gunakan browser developer tools untuk melihat bagaimana kode bekerja.
